## Part A - Median Age Computation

### Objective
Create a user-defined function (`MedianAge`) to compute the median passenger age, automatically replacing null entries with `23` during the calculation.

### Solution strategy
- Implemented using a PySpark `@pandas_udf` (Pandas User-Defined Function).
- Modern PySpark uses Apache Arrow to vectorize operations. A Pandas UDF is significantly more memory-efficient and faster for aggregations than standard row-by-row Python UDFs.
- Handled nulls internally using Pandas `.fillna(23)` before applying the median calculation.

### Key results
- Calculated Median Age: `24.0`

## Part B - Filtering Passengers

### Objective
Select the `Name` and `Fare` of all passengers whose age is strictly greater than the computed median age from Part A, using only built-in DataFrame/SQL operations.

### Solution strategy
- Used the DataFrame API `filter()` and `select()` methods.
- Applied the `coalesce` function to ensure consistency with Part A: any null ages were evaluated as `23` during the filtering condition to prevent missing data from causing false negatives in the comparison.

### Key results
- The query successfully returns a filtered DataFrame. 
- Output displays the top 20 rows of passengers strictly older than 24.0.

## Part C - Hypothesis Verification

### Objective
Verify the hypothesis: *"The average fare price of passengers who survived the sinking of the Titanic is not very different from the average fare price of passengers who did not survive."*

### Solution strategy
- Used PySpark DataFrame aggregations: `groupBy("Survived").agg(avg("Fare"))`.
- Evaluated the resulting averages using a simple numeric comparison as permitted by the exercise instructions.

### Key results
- Average fare for Survivors (`Survived = 1`): `~48.40`
- Average fare for Non-Survivors (`Survived = 0`): `~22.12`

### Conclusion
**The hypothesis is REJECTED.** There is a significant and clearly observable difference between the two groups. Survivors paid, on average, more than double the fare of non-survivors. Therefore, the statement that the average fares are "not very different" is demonstrably false.
